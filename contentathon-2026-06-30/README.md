# RSQKit Contentathon 2026-06-30

## Team
Shoaib Sufi, Laura Portell, Aleks Nenadic, Michael Sparks

## Aims

The aim of the contentathon is to create drafts of task pages for inclusion into RSQKit and to make pull requests for the RSQKit Editorial Board to review. 

Drafts can either be created manually from scratch or with the help of an AI (based on authors preferences), each option following task page structure and guidelines.
